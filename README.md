# SEMANAL TC

O aplicativo foi desenvolvido na linguagem Python versão 3.10.6. O objetivo geral do SEMANAL TC é padronizar e otimizar a realização e as análises dos testes semanais de tomografia computadorizada por meio de uma interface fácil e intuitiva para o usuário. 


---
## Utilizar o software (Windows)

Certifique-se de ter o Python e o pip instalados em seu sistema. Em seguida, execute o seguinte comando para instalar as dependências necessárias:

```pip install -r requirements.txt```
